# Questão 2
Escreva uma função recursiva que receba um número inteiro, não negativo, e retorne se ele é primo.

Assinatura da função: def primo(n)

Dica 01: É necessário criar uma função auxiliar que receba dois parâmetros: o número a ser testado e um divisor
para testar se o número é divisível por ele.
Dica 02: O caso base ocorre quando o divisor é igual a 1. Nesse caso o número é primo.
